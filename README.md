# 👻 GhostPeek

Peek Discord DMs without marking them as read.

GhostPeek lets you instantly preview messages in a clean popup using **right-click** or **Ctrl+click**, while keeping your read status untouched.

---

## ✨ Features

* 👀 **Peek messages without marking as read**
* ⚡ **Instant loading with smart caching**
* 🔍 **Built-in search with highlight & navigation**
* 📜 **Background loading of full message history**
* 📎 **Supports images, videos, embeds, and attachments**
* 📞 **Call duration tracking per conversation**
* 🖱️ **Draggable & resizable popup UI**
* 🔒 **Completely non-intrusive (read-state safe)**

---

## 📦 Installation

### 1. Install BetterDiscord

Download and install BetterDiscord:
👉 https://betterdiscord.app/

### 2. Download GhostPeek

* Go to the **Releases** page
* Download the latest `GhostPeek.plugin.js`

### 3. Move to Plugins Folder

Place the file in your BetterDiscord plugins folder:

```
Windows: %appdata%/BetterDiscord/plugins
Mac: ~/Library/Application Support/BetterDiscord/plugins
Linux: ~/.config/BetterDiscord/plugins
```

### 4. Enable Plugin

* Open Discord
* Go to **Settings → Plugins**
* Enable **GhostPeek**

---

## 🧠 Usage

* **Right-click a DM → "Peek Messages"**
* OR **Ctrl + Click a DM**

A popup will appear showing messages without marking them as read.

---

## ⚙️ Settings

* **Cache Duration** – Control how long messages are cached
* **Call Time Format** – Customize how call durations are displayed
* **Data Management** – Clear cache or reset call tracking
